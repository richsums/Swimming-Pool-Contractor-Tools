    About PoolBid Designer

    PoolBid Designer is a bid-phase design assistant for California pool contractors. It transforms a hand sketch and a property address into a structured parametric model — then generates a professional preliminary drawing package and a separate California concern review.

    Workflow

    1. Project Setup — capture client and address.
    2. Sketch — upload a sketch and confirm interpreted features.
    3. Site & Aerial — geocode the lot, place the pool on the aerial map.
    4. Design — edit pool shell, features, decking, safety features, and setbacks.
    5. 3D View — orbit the model for the customer hero rendering.
    6. Concerns — review findings for safety, setbacks, drainage, AHJ coverage.
    7. Export — generate the bid package PDF + a separate internal concern PDF.

    Keyboard shortcuts

    • Esc — close modals
    • Alt + ← / → — move between workflow steps
    • Ctrl/⌘ + S — save current revision

    Important limitations

    This is a preliminary/bid-phase tool. It does not produce engineered or permit-ready drawings. Final approval remains with the local authority having jurisdiction. The application uses heuristic sketch interpretation; in production this is replaced by a trained CV pipeline. Aerial imagery shown is from OpenStreetMap-licensed sources and should be verified for currency before customer presentation.
